# Foundation Lab 002: Wireshark Core Features

Dieses Lab trainiert wichtige Wireshark-Funktionen, die in reinen Protokollkapiteln leicht zu kurz kommen.

Du arbeitest mit einer vorhandenen lokalen PCAP-Datei und nutzt Funktionen wie:

- Capture File Properties
- Find Packet
- Packet Comments
- File Comment
- Time Reference
- Mark Packets
- Export Specified Packets
- Ignore Packets
- Decode As

## Lab-Pfad

```text
labs/foundation/lab-foundation-002-wireshark-core-features/
```

## Ziel

Du übst:

- eine PCAP-Datei als Analyseobjekt vorzubereiten
- wichtige Datei- und Paketinformationen zu finden
- relevante Pakete zu markieren und zu kommentieren
- einen kleineren Ausschnitt zu exportieren
- Zeitbezug mit Time Reference herzustellen
- Wireshark-Funktionen methodisch und nicht zufällig zu verwenden

## Voraussetzungen

Du brauchst eine vorhandene lokale PCAP-Datei.

Empfohlen:

```text
pcaps/generated/lab-basic-020-dns-http-docker.pcapng
```

oder:

```text
pcaps/generated/lab-basic-030-tcp-handshake-reset.pcapng
```

Falls keine vorhanden ist, bearbeite zuerst eines der vorherigen Labs.
