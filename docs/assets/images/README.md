# Images

Dieser Ordner enthält Bilder und Screenshots für die Kursdokumentation.

## Struktur

```text
docs/assets/images/
├── wireshark-ui/
├── display-filter/
├── tshark/
├── tcp/
├── dns/
├── http-tls/
├── performance/
├── security/
├── labs/
└── wca-prep/
```

## Regeln

- Nur Lab-Daten oder saubere Testdaten verwenden.
- Keine produktiven Captures.
- Keine personenbezogenen Daten.
- Keine Zugangsdaten, Tokens oder Cookies.
- PNG für UI-Screenshots bevorzugen.
- Aussagekräftige Dateinamen verwenden.
- Jeden Screenshot mit Alt-Text einbinden.
- Vor dem Commit den Screenshot-Styleguide prüfen.

```text

```
