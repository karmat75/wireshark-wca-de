# WCA-Practice-Quiz

Das WCA-Practice-Quiz ist eine gemischte Wiederholung.

Es enthält szenariobasierte Fragen über Foundation, Basis-Kurs, erweiterten Kurs, Labs und Methodik.

## Datei

```text
quizzes/questions/wca-practice.yml
```

## Aktueller Umfang

```text
15 Fragen
```

## Themen

- Wireshark-Funktionen
- Capture-Punkt
- Display Filter
- DNS
- ARP
- TCP
- TLS
- HTTP
- Performance
- Methodik
- Security-Einordnung

## Nutzung

```bash
python3 tools/wwca/wwca.py quiz run wca-practice --shuffle
```

## Ziel

Dieses Quiz soll nicht auswendig gelernte Definitionen prüfen, sondern WCA-nahe Denkweise:

- Szenario lesen
- Schicht einordnen
- Beobachtung bewerten
- Antwort nicht überziehen
- Capture-Punkt beachten
- nächste sinnvolle Prüfung ableiten

## Hinweis

Diese Fragen sind eigene Kursfragen.

Sie sind keine echten WCA-Prüfungsfragen und keine Exam Dumps.
