# Lab Basic 010: Display Filter

## Ziel

In diesem Lab trainierst du Display Filter systematisch.

Du arbeitest mit DNS-, HTTP- und TCP-Paketen aus einer kontrollierten Docker-Lab-Umgebung.

## Voraussetzungen

Du brauchst:

- Wireshark
- TShark
- Docker und Docker Compose, wenn du die PCAP neu erzeugst
- eine lokale PCAP aus dem DNS/HTTP-Lab

Empfohlene Datei:

```text
pcaps/generated/lab-basic-020-dns-http-docker.pcapng
```

Falls diese Datei nicht existiert, kannst du in den Aufgaben eine neue PCAP erzeugen.

## Ergebnis

Am Ende hast du:

- eine Liste wichtiger Display Filter
- eine Tabelle mit Trefferzahlen
- mehrere belegte Beobachtungen
- TShark-Ausgaben zur Gegenprüfung
- ein besseres Gefühl für Filter-Fallen

## Geschätzte Dauer

```text
45 Minuten
```
