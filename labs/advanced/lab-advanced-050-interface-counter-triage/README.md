# Lab Advanced 050: Interface Counter Triage

## Ziel

Dieses Lab trainiert die Einordnung von Interface-Countern aus einem Monitoring-System.

Du entscheidest, ob Wireshark sinnvoll helfen kann und welche Schritte in der realen Welt zuerst kommen.

## Voraussetzungen

Du brauchst:

- Grundverständnis von Switch-Interfaces
- TCP Retransmissions und Duplicate ACKs aus den vorherigen Labs
- Wissen über Capture-Punkte
- keine Docker-Umgebung
- keine PCAP-Datei

## Ergebnis

Am Ende hast du:

- eine Counter-Triage erstellt
- sinnvolle nächste Schritte definiert
- Capture-Punkte vorgeschlagen
- Grenzen von Wireshark benannt
- eine kurze, vorsichtige Bewertung geschrieben

## Geschätzte Dauer

```text
45 Minuten
```

## Wichtiger Punkt

Dieses Lab zeigt nicht, wie man „den richtigen Filter“ findet.

Es zeigt, wie man entscheidet, ob ein Filter überhaupt die passende Antwort ist.
