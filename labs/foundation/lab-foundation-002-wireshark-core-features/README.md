# Lab Foundation 002: Wireshark Core Features

## Ziel

Dieses Lab trainiert zentrale Wireshark-Funktionen, die für Praxis und WCA-Vorbereitung wichtig sind.

Es geht nicht um ein neues Protokoll, sondern um saubere Bedienung und Dokumentation.

## Voraussetzungen

Du brauchst:

- Wireshark
- TShark
- eine vorhandene lokale PCAP-Datei aus einem vorherigen Lab

Empfohlen:

```text
pcaps/generated/lab-basic-020-dns-http-docker.pcapng
```

oder:

```text
pcaps/generated/lab-basic-030-tcp-handshake-reset.pcapng
```

Wenn keine Datei vorhanden ist, erzeuge zuerst einen Capture aus einem vorherigen Lab.

## Ergebnisdateien

In diesem Lab können lokale Arbeitsdateien entstehen:

```text
pcaps/generated/lab-foundation-002-selected-packets.pcapng
```

Optional:

```text
pcaps/generated/lab-foundation-002-with-comments.pcapng
```

Diese Dateien sollen nicht committed werden.

## Geschätzte Dauer

```text
45 Minuten
```

## WCA-Bezug

Dieses Lab übt:

- Wireshark Core Features
- Capture File Properties
- Find Packet
- Packet Comments
- File Comment
- Export Specified Packets
- Time Reference
- Mark und Ignore Packets
- vorsichtige Decode-As-Einordnung
