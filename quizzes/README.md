# Quizzes

Dieser Ordner enthält maschinenlesbare Quizfragen für den Kurs.

## Struktur

```text
quizzes/
├── questions/
│   ├── foundation.yml
│   └── basic.yml
├── exams/
│   └── .gitkeep
└── objectives/
    └── .gitkeep
```

## Grundsatz

Die Fragen sind eigene Kursfragen.

Sie sind keine echten WCA-Prüfungsfragen und keine Exam Dumps.

## Spätere Nutzung

Geplant ist ein lokales Tool, das diese Fragen ausführt und Fortschritt lokal speichert.
