# Challenge 001: DNS/HTTP Triage

## Kurzbeschreibung

Ein Client greift in einer kleinen Lab-Umgebung auf interne Webserver zu.

Ein Name existiert nicht, ein Webserver antwortet normal und ein Webserver antwortet langsam.

Deine Aufgabe ist es, den Capture zu analysieren und eine kurze, belegbare Bewertung zu schreiben.

- Wireshark
