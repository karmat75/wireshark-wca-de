# Challenge 002: TCP Handshake und Reset

## Kurzbeschreibung

Ein Client erreicht einen Webserver auf Port 80.

Ein zweiter Verbindungsversuch auf Port 81 schlägt fehl beziehungsweise wird aktiv abgelehnt.

Deine Aufgabe ist es, die TCP-Situationen im Capture zu unterscheiden und sauber zu bewerten.
