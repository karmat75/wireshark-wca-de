# Paket 18: Lab-Konzept und erstes Foundation-Lab

Dieses Paket ergänzt den Kurs um die erste praktische Lab-Struktur.

```text

```

## Ziel

Mit diesem Paket beginnt der Kurs, von Dokumentation zu einem echten Trainingssystem zu werden.

Enthalten sind:

- eine MkDocs-Seite für Labs und Übungen
- eine einheitliche Lab-Struktur
- das erste Foundation-Lab
- eine maschinenlesbare `metadata.yml`
- Aufgaben, Hinweise und Musterlösung

Das erste Lab erzeugt noch keine bereitgestellte PCAP-Datei, sondern lässt Lernende bewusst selbst einen ersten Capture erstellen.
