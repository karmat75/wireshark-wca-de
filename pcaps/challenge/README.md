# Challenge PCAPs

Dieser Ordner ist für feste, veröffentlichbare Challenge-PCAPs vorgesehen.

Aktuell werden die ersten Challenges bewusst lokal aus Docker-Labs erzeugt.

## Regeln

- Keine produktiven Captures.
- Keine personenbezogenen Daten.
- Keine Zugangsdaten, Cookies oder Tokens.
- Nur synthetische oder klar lizenzierte Captures.
- Herkunft und Lizenz müssen dokumentiert sein.
- Große Dateien nur bewusst aufnehmen.

Lokal erzeugte Captures gehören nach:

```text
pcaps/generated/
```
